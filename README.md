# Suricata IDS Rule Tuning

## Project Title
Rule Tuning in Suricata IDS to Reduce False Positives and Alert Noise

## Description
This project demonstrates how to reduce excessive alerts and false positives in Suricata using Suppress and Threshold techniques. Custom detection rules were also created and tested.

## Objectives
- Reduce alert noise using rule tuning
- Create custom rules for detection
- Show improvement before and after tuning

## Files Included
- [local-rules](suricata-rules-tuning/rules/local.rules) → Custom detection rules
- [threshold.config-file](suricata-rules-tuning/config/threshold.config) → Suppress and Threshold rules
- `CA_Report.pdf` → Final report with screenshots
